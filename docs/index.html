Load Your Images Lazily

A simple, vanilla JS library that loads images on your page only when they scroll into view.

Getting Started

  • cdn
  • npm
  • yarn
  • express
  • react

cdn

Using a CDN link is quick and easy â€” plus, you don't have to download anything. Just copy the following code into the <head> section of your webpage.

            
  <script src="https://cdn.jsdelivr.net/gh/dreamistlabs/imagezy/dist/imagezy.min.js"></script>
  <link href="https://cdn.jsdelivr.net/gh/dreamistlabs/imagezy/dist/imagezy.css" rel="stylesheet">

npm

You can install the library and styles via npm in the command line.

            
  npm install imagezy --save

yarn

You can install the library and styles via yarn in the command line.

            
  yarn add imagezy

express

Make sure you've added Imagezy to your react application, either with npm or yarn, via the command line.

            
  npm install imagezy --save

  /* or */

  yarn add imagezy

Next, add the following to your app.js file:

            
  app.use('/imagezy', express.static(__dirname + '/node_modules/imagezy/dist/'));

Then, in your project's main layout view, add the following code:

            
  <script src='/imagezy/imagezy.min.js'></script>
  <link href='/imagezy/imagezy.css' rel='stylesheet'>

react

Make sure you've added Imagezy to your react application, either with npm or yarn, via the command line.

            
  npm install imagezy --save

  /* or */

  yarn add imagezy

Next, add the following to your App.js file:

            
  import 'imagezy/dist/imagezy.min.js';
  import 'imagezy/dist/imagezy.css';
